# 04 · שלד לוח הסידור (Board Skeleton — read-only)

**מטרה:** לצייר את הטבלה הריקה כמו בתמונה — עמדות (שורות) × ימים (עמודות), מחולק ל-3 משמרות —
לפי הפרופיל שנבחר לשבוע. עדיין בלי שמות מאבטחים. מתחילים מהקונספט ב-`design-system/concepts/schedule-builder/index.html`.

**מה נבנה:**
- שיוך **פרופיל↔שבוע** (איזה פרופיל השבוע משתמש בו) — מודל/שדה + בורר.
- קומפוננטת לוח (board) שמרנדרת עמדות×ימים×משמרות מתוך הפרופיל; מכבדת ימים פעילים פר-עמדה.
- ראוט באדמין, קריאה בלבד.

**קריטריון הצלחה (המשתמש מריץ):** פותח בונה לשבוע הבא → רואה גריד תואם בדיוק לעמדות שהגדיר
ב-03, בחלוקה בוקר/ערב/לילה, עם תאים ריקים בימים הפעילים בלבד.

**קבצים צפויים:** שיוך פרופיל↔שבוע (מיגרציה/שדה), `frontend/admin/src/pages/ScheduleBuilderPage.jsx`,
`components/board/*` (פורט מ-`index.html`).

**תלוי ב:** 02, 03.
