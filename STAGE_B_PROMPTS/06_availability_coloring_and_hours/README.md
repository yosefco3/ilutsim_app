# 06 · חיבור לזמינות אמיתית + צביעה + שעות שנותרו

**מטרה:** לחבר את הלוח ל**זמינות האמיתית** (שנכנסה במשימה 01 למודל הזמינות), לצבוע על הלוח איפה
המאבטח הנבחר זמין, ולמקד את העבודה לפי **שעות שנותרו**.

**מה נבנה:**
- שירות שמושך זמינות מ-`WeeklySubmission`/`DailyStatus`/`ShiftWindow` לשבוע ולמיר אותה לחלונות פר-יום.
- **כלל האיחוד** (קריטי): שעות פנויות = משך **איחוד** החלונות היומיים, לא סכומם
  (`07:00–16:30` ∪ `15:00–19:00` = 12ש', לא 13.5). פורט מתוקן של `logic.js::weeklyBudget`.
- צביעת זמינות המאבטח הנבחר על הלוח; "שעות שנותרו" מתעדכן עם כל שיבוץ; מיון הרשימה לפי שנותר.

**קריטריון הצלחה (המשתמש מריץ):** בוחר מאבטח שהגיש זמינות → רואה את חלונותיו נצבעים נכון על הלוח;
משבץ אותו → "שעות שנותרו" יורד בכמות הנכונה (לפי האיחוד); הרשימה ממוינת מהפנוי-ביותר לפנוי-הפחות.

**קבצים צפויים:** `services/availability_service.py`, פורט `logic.js`, חיבור ל-board.

**תלוי ב:** 01 (דאטא זמינות), 04, 05.

> פרומפטים מלאים ייכתבו כשנגיע למשימה זו.
