# 07 · אזהרות רכות חיות (+ מאפייני מאבטח)

**מטרה:** להתריע (בלי לחסום) על שיבוצים בעייתיים, כמו בצד המאבטח. כאן גם נכנסים **מאפייני המאבטח**,
כי בלעדיהם אי-אפשר לבדוק התאמת-מאפיין.

**מה נבנה:**
- **מאפייני מאבטח** (חמוש / רכב עירייה / רוני / הליכה מרובה...) על מודל `User` + מיגרציה + טופס עריכה.
  > ⚠️ אפיון המאבטחים עדיין לא נסגר אצל המשתמש — לאשר את הרשימה הסופית **לפני** הקיבוע.
- פורט `logic.js::warnings` — סוגי אזהרות: מחוץ-לזמינות, כיסוי חלקי, חוסר-מאפיין, שיבוץ-כפול
  (שעות חופפות), עמדה ריקה, חריגה ממנוחה/ימים רצופים.
- הצגה צבעונית ואינפורמטיבית על הלוח; **השיבוץ תמיד מותר**.

**קריטריון הצלחה (המשתמש מריץ):** משבץ מאבטח מחוץ לזמינותו → אזהרה צהובה אך השיבוץ עובר;
משבץ אותו בעמדה שדורשת "חמוש" כשאינו חמוש → אזהרת חוסר-מאפיין; שיבוץ כפול חופף → אזהרת כפילות.

**קבצים צפויים:** `User` (מאפיינים) + מיגרציה, טופס עריכת מאבטח, פורט `logic.js`, שכבת אזהרות ב-board.

**תלוי ב:** 05, 06; אישור אפיון המאבטחים מהמשתמש.
