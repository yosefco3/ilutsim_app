# 09 · חריגים פר-שבוע (Per-week Overrides)

**מטרה:** גמישות פר-שבוע בלי לשנות את הפרופיל הקבוע — הקופסאות הצבעוניות בצד התמונה
(טקס קריאת רחוב, ועדת תכנון ובנייה, מר ברונר, זמנים). הוספה/ביטול עמדה לשבוע בודד.

**מה נבנה:**
- מנגנון override ברמת השבוע: עמדה שמתווספת לשבוע ספציפי, או עמדה מהפרופיל שמבוטלת לשבוע מסוים.
- UI להוספת/ביטול עמדה לשבוע הנוכחי בלבד.

**קריטריון הצלחה (המשתמש מריץ):** מוסיף עמדת אירוע לשבוע אחד → היא מופיעה בלוח **רק** באותו שבוע,
ולא בשבועות אחרים; מבטל עמדה קבועה לשבוע מסוים → היא נעלמת רק שם.

**קבצים צפויים:** מודל/שדה override (מיגרציה), service, חיבור UI ל-board.

**תלוי ב:** 03, 04.

> פרומפטים מלאים ייכתבו כשנגיע למשימה זו.
