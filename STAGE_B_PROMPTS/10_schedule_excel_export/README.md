# 10 · ייצוא הסידור לאקסל (Schedule Excel Export)

**מטרה:** להוציא את הסידור הבנוי לאקסל **בפורמט המוכר** של התמונה (עמדות × ימים, 3 משמרות,
שמות מאבטחים בתאים עם שעות מותאמות, קופסאות אירועים, קוד צבעים).

**מה נבנה:**
- הרחבת `ExcelExportService` בדוח סידור: שורות = עמדות (לפי משמרת), עמודות = ימים, תאים = מאבטח+שעות.
- קוד צבעים תואם (משמרות, תאים מיוחדים), קופסאות חריגים פר-שבוע (משימה 09).
- Endpoint `GET /admin/export/schedule/{week_id}` + כפתור ייצוא בבונה.

**קריטריון הצלחה (המשתמש מריץ):** לוחץ "ייצא סידור" → מקבל xlsx שנראה כמו התמונה הידנית,
עם השיבוצים, השעות וההערות הנכונים.

**קבצים צפויים:** `services/excel_export_service.py` (דוח חדש), controller, כפתור UI.

**תלוי ב:** 05, 09.

> פרומפטים מלאים ייכתבו כשנגיע למשימה זו.
