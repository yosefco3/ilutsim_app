# 08 · גרירה + מיון אוטומטי (Drag & Drop)

**מטרה:** חוויית העבודה המלאה מהקונספט — גרירת מאבטח לתא, מיון אוטומטי של רשימת המאבטחים לפי
שעות שנותרו, וזרימת-המשנה (קליק-ימני על תא ריק → מועמדים זמינים). פורט אינטראקציות מ-`index.html`.

**מה נבנה:**
- Drag & drop של מאבטח מהרשימה לתא בלוח (כותב שיבוץ דרך API של 05).
- מיון אוטומטי חי של הרשימה; הסרה ידנית של מאבטח מהרשימה (קליק-ימני / מקש F).
- זרימת-משנה: קליק-ימני על תא ריק → רשימת מועמדים זמינים לאותה משבצת.

**קריטריון הצלחה (המשתמש מריץ):** גורר מאבטח לתא → נשמר ונצבע; הרשימה מתעדכנת ומסתדרת לבד;
קליק-ימני על תא ריק → מקבל רשימת זמינים; מסיר מאבטח מהרשימה ידנית.

**קבצים צפויים:** `components/board/*` (גרירה), אינטראקציות מקלדת/עכבר, חיבור ל-05/06.

**תלוי ב:** 05, 06, 07.
