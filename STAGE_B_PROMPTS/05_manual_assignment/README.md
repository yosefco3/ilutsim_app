# 05 · שיבוץ ידני (Manual Assignment)

**מטרה:** למלא את התאים — לשבץ מאבטח לעמדה×יום×שבוע, ולבטל שיבוץ. השכבה שהופכת את השלד לסידור.

**מה נבנה:**
- מודל `ScheduleAssignment` (+ מיגרציה) — שבוע, עמדה, יום, מאבטח, ושעות מותאמות אופציונליות
  (דריסת שעות פר-תא, כמו "מאי עד 19" בתמונה).
- API: assign / unassign / רשימת שיבוצים לשבוע (`require_admin_role`).
- חיבור בסיסי ב-UI (בחירת מאבטח לתא; ללא גרירה/אזהרות עדיין).

**קריטריון הצלחה (המשתמש מריץ):** משבץ מאבטח לתא → מרענן → השיבוץ נשמר; מבטל שיבוץ → נעלם.
שיבוץ עם שעות מותאמות נשמר עם השעות.

**קבצים צפויים:** `models/schedule_assignment.py`, מיגרציה, repo/service/controller, schemas, חיבור UI.

**תלוי ב:** 03, 04.
