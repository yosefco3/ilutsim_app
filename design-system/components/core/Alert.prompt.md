One-line: inline, soft-tinted feedback banner shown inside forms and pages.

```jsx
<Alert tone="error">שם משתמש או סיסמה שגויים</Alert>
<Alert tone="success">ההגדרות נשמרו בהצלחה</Alert>
<Alert tone="warning">שים לב — ייתכן שחרגת מהכללים (ניתן לשלוח בכל זאת)</Alert>
```

Tones: `error` / `success` / `warning` / `info`. Use for form-level messages and the brand's signature *soft, non-blocking* constraint warnings. For transient global notices use Toast instead.
